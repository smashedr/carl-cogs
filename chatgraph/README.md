# ChatGraph

Generate Pie Graph of Messages in Current or Specified Channel.

**Web API:** Cog _may_ require the Web API for additional functionality. [API Setup...](../README.md#web-api)

## Install

```text
[p]cog install carl-cogs chatgraph
[p]load chatgraph

[p]help ChatGraph
```
