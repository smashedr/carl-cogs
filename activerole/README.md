# ActiveRole

Adds a role to active chatters.

**Requires Redis:** Cog requires Redis to function. [Redis Setup...](../README.md#redis)

## Install

```text
[p]cog install carl-cogs activerole
[p]load activerole

[p]help ActiveRole
```
